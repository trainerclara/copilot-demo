
# Facilitator Guide (60 minutes)

**0–5 min** — Framing: What Copilot is (inline, Chat, PR review). Safety: "Copilot suggests; you decide."

**5–15 min** — Inline generation: `calculator.py`. Show cycling suggestions and partial accept.

**15–25 min** — Explain & Refactor: select code → Chat → explanations, docstrings, logging.

**25–35 min** — Tests: generate pytest tests in `test_calculator.py`, run `pytest`, fix failures.

**35–45 min** — Data: complete `data_tasks.py` with `pandas`, print/plot summary.

**45–55 min** — Docs: draft README sections and PR template using Chat.

**55–60 min** — Q&A and responsible use recap. Toggle Copilot, report poor completions.

**Trainer tips**
- Type slowly to let inline suggestions appear.
- Select code before asking in Chat to boost context.
- Celebrate failures: use them to show iteration and testing.
- Keep prompts specific: inputs, outputs, constraints.
