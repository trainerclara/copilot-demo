
# Exercise 05 — Docs & Repo Hygiene

Open `README.md` and ask Chat to generate a concise **Getting Started** section for newcomers.

Create a PR template in `.github/PULL_REQUEST_TEMPLATE.md` with sections:
- Summary
- What changed
- How to test
- Screenshots
- Checklist
- Linked issues

Optionally create an issue template under `.github/ISSUE_TEMPLATE`.
